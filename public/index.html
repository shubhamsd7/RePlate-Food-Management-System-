ℹ️ About FoodRescue

🎯 The Problem

Restaurants waste tons of surplus food every day while shelters struggle to feed people in need. Traditional donation systems are slow, inefficient, and lack real-time coordination.

💡 Our Solution

FoodRescue is a lightweight web app that instantly matches nearby shelters with restaurants offering surplus meals. Real-time SMS notifications, gamification, and carbon tracking make food rescue fast, engaging, and impactful.

🚀 Key Features

  • Real-Time Matching: Instant connection between restaurants and shelters
  • SMS Notifications: Twilio integration for immediate alerts
  • Location-Based: Google Maps API finds nearest shelters automatically
  • Carbon Tracking: See your environmental impact with every donation
  • Gamification: Leaderboard system rewards active shelters
  • Impact Dashboard: Visualize meals saved and CO₂ prevented
  • Mobile-Friendly: Works on any device

🛠️ Tech Stack

  • Node.js + Express backend
  • Google Maps API for geolocation
  • Twilio SMS for instant notifications
  • Firebase-ready architecture (can be integrated)
  • Responsive vanilla JS frontend
  • RESTful API design

🏆 Hackathon-Winning Features

  • AI-Ready: Architecture supports future AI food classification
  • Blockchain-Ready: Can integrate transparent donation tracking
  • Behavioral Psychology: Gamification drives engagement
  • Real Impact Metrics: Quantifiable carbon savings
  • Scalable Design: Easily handles multiple cities
  • Social Good: Addresses UN Sustainable Development Goals

🎨 Future Enhancements

  • Computer vision for automatic food quantity estimation
